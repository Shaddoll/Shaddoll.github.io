Virtual Memory in Xv6

Zijian Chen

In this project, I implemented virtual memory in MIT's Xv6 Operating System by modifying the system's kernel code. I allocated a swap area in the disk and designed a data structure to manage the swap area, and I also implemented the page fault handling function and a simple FIFO page replacement algorithm.

Real-time Environmental Sensing System

Zijian Chen, Haoci Zhang

In this project, we developed a real-time environmental sensing system which could visualize environmental data on PCs in real-time. We used three sensors. One of them was used for receiving packets and forwarding packets to the computer through USB port, and the other two is used to collect data. The program controlling the sensors was written in NesC based and ran on TinyOS, and the program visualizing environmental data is written in Java.
